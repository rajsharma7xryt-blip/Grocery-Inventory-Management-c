# Grocery Inventory Management System

A simple menu-driven Grocery Inventory Management System developed in C. This project allows users to manage grocery products by adding, displaying, searching, updating, and deleting product records.

## Features

- Add new products
- Display all products
- Search a product by ID
- Update product quantity
- Delete a product
- Menu-driven interface

## Technologies Used

- C Programming
- Structures
- Arrays
- Functions
- Loops
- Conditional Statements

## How to Run

1. Compile the program:
   ```bash
   gcc main.c -o grocery
   ```

2. Run the executable:
   ```bash
   ./grocery
   ```

   **For Windows:**
   ```bash
   grocery.exe
   ```
